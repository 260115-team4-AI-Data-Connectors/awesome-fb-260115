- Git 
- Git Bash 
- Choice of Editor 
- Python^3.9

## Usage

### Linux(Tested on Debian 13)

1. Open up bash(or terminal of your choice)

2. Get to directory of your choice & Execute following commands in sequence
```bash
$ git clone https://github.com/260115-team4-AI-Data-Connectors/awesome-fb-260115
$ cd awesome-fb-260115
$ python3 fizzbuzz.py # To try fizzbuzz out
$ python3 main.py # To try 100 Prisioner Problems
```
3. enjoy results

### Windows 11
1. Open up powershell
2. get to directory of your choice & Execute following commands in sequence
```powershell
$ git clone https://github.com/260115-team4-AI-Data-Connectors/awesome-fb-260115
```
